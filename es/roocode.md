# Usar AICentOS con RooCode

## Instalación en VSCode

1. Busca [RooCode](https://marketplace.visualstudio.com/items?itemName=RooVeterinaryInc.roo-cline) en VSCode e instálalo
2. Visita [https://www.aicentos.com/console/token](https://www.aicentos.com/console/token) para obtener tu API Key

## Agregar Proveedor

Abre RooCode y agrega un proveedor compatible con OpenAI:

| Configuración | Valor |
|---------------|-------|
| **URL base de OpenAI** | `https://www.aicentos.com/v1` |
| **API Key** | Tu clave de AICentOS |
| **Modelo** | `gpt-5` |

::: tip Consejo
También puedes usar `glm-4.5`, `glm-4.6`, o `deepseek-v3.1`.
:::

Guarda y comienza a usar.
