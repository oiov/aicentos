# Usar NBility en Cursor

> ¿Primera vez? Completa el [registro de cuenta](/es/account) y genera tu API Key en la consola.

## 1. Abrir la configuración de Cursor

Abre Cursor y entra en configuración.

## 2. Ir a Models

En el menú lateral, haz clic en **Models**.

## 3. Añadir un modelo y completar API Key / Base URL

Haz clic en **Añadir modelo** y completa:

- **API Key**: clave de NBility
- **Base URL**: `https://nbility.dev/v1`

![Ejemplo de configuración de Cursor Models](/img/start/cursor.png)

Referencias:

1. Entrada de configuración
2. Menú Models
3. Botón de añadir modelo
4. Campo API Key
5. Campo Base URL

## 4. Guardar y verificar la conexión

1. Crea una conversación
2. Selecciona el modelo habilitado
3. Envía un prompt de prueba
4. Respuesta normal = conexión correcta
