# Utiliser AICentos dans Cursor

> Première utilisation ? Terminez la [création de compte](/fr/account) puis générez votre API Key dans la console.

## 1. Ouvrir les paramètres de Cursor

Ouvrez Cursor puis cliquez sur l’icône de paramètres.

## 2. Accéder à Models

Dans le menu latéral, cliquez sur **Models**.

## 3. Ajouter un modèle et renseigner API Key / Base URL

Cliquez sur **Ajouter un modèle**, puis renseignez :

- **API Key** : clé AICentos
- **Base URL** : `https://aicentos.com/v1`

![Exemple de configuration Cursor Models](/img/start/cursor.png)

Repères :

1. Entrée paramètres
2. Menu Models
3. Bouton d’ajout
4. Champ API Key
5. Champ Base URL

## 4. Enregistrer et vérifier la connexion

1. Créez une conversation
2. Sélectionnez le modèle activé
3. Envoyez un prompt de test
4. Réponse normale = connexion OK
