# 账户注册

> 已有账号？直接查看 [快速开始](/start) 配置工具。

## 一、获取 API Key

### 1. 注册账号

访问 [nbility.dev](https://nbility.dev/register?aff=Dptp)，点击**注册**：



选择注册方式（GitHub、LinuxDO 或用户名注册）：



填写用户名、密码和确认密码完成注册：



### 2. 登录

注册完成后使用用户名和密码登录：



登录后进入控制台：



### 3. 创建令牌

进入**控制台 → 令牌管理 → 添加令牌**，填写表单：



- **令牌分组**建议选择 **aws-enterprise**，该分组包含 `claude-opus-4-5-20251101`、`claude-haiku-4-5-20251001`、`claude-opus-4-6`、`claude-sonnet-4-5-20250929`、`claude-sonnet-4-6` 等模型。模型可用性 https://nbility.dev/status

创建完成后在令牌列表点击**复制**按钮获取 API Key（格式为 `sk-xxx`）：



### 4. 充值

进入**控制台 → 钱包管理**，支持支付宝、微信或兑换码：


| 方式 | 操作路径 |
|---|---|
| 支付宝 | 输入/选择金额 → 支付宝 |
| 微信 | 输入/选择金额 → 微信 |
| 兑换码 | 兑换码区域输入 → 点击兑换额度 |

::: tip 倍率说明
分组名称中的 `0.5x倍率` 表示消费倍率，数值越小获得更多 Token，如 `0.5x` 即 1 元可使用官方 1000 万 Token（官方原价 500 万）。
:::
