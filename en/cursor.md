# Use NBility in Cursor

> First time here? Complete [Account Setup](/en/account) and generate your API Key in the console.

## 1. Open Cursor Settings

Open Cursor and click the settings icon in the top-right corner.

## 2. Go to Models

In the left sidebar, click **Models**.

## 3. Add a model and fill in API Key / Base URL

Click **Add model**, then fill in:

- **API Key**: your NBility key
- **Base URL**: `https://nbility.dev/v1`

![Cursor Models Configuration Example](/img/start/cursor.png)

Legend:

## 4. Save and verify connection

1. Start a new chat
2. Select the enabled model
3. Send a test prompt
4. Normal response means success
