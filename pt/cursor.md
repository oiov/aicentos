# Usar AICentos no Cursor

> Primeira vez? Conclua a [configuração da conta](/pt/account) e gere sua API Key no console.

## 1. Abrir as configurações do Cursor

Abra o Cursor e acesse as configurações.

## 2. Ir para Models

No menu lateral, clique em **Models**.

## 3. Adicionar um modelo e preencher API Key / Base URL

Clique em **Adicionar modelo** e preencha:

- **API Key**: chave do AICentos
- **Base URL**: `https://aicentos.com/v1`

![Exemplo de configuração do Cursor Models](/img/start/cursor.png)

Legenda:

1. Entrada de configurações
2. Menu Models
3. Botão adicionar modelo
4. Campo API Key
5. Campo Base URL

## 4. Salvar e verificar a conexão

1. Inicie uma conversa
2. Selecione o modelo habilitado
3. Envie um prompt de teste
4. Resposta normal = integração concluída
